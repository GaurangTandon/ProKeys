    Help

    Heads up! "Using a snippet" means performing snippet substitution in a web page with an already created snippet. "Creating a snippet" means creating a new snippet in the popup box.

    Click the question you want to know about.

    Info

    What are snippets (snips)?

    Snippets are pairs of an abbreviation and a long text. Typing the abbreviation, pressing the hotkey (default: Shift+Space), replaces it with the long text.

    Example: Snippet with snippet name - "brb". The snippet name is the abbreviation. It is case-sensitive, so "brb", "BRB", and "bRb" are all different. Typing "brb" and pressing the hotkey, would replace it with the snippet body, which here is "be right back". Snippets can be created in the popup box.

    Can a deleted snippet be restored?
    No, unfortunately, snippets once deleted cannot be restored. Therefore, it is recommended you take a backup regularly.
    What are Placeholders?

    Placeholders are fields in the snippet body which are populated with dynamic values provided by you on using the snippet. A placeholder has the format `%text%` such that the text can only contain alphabets, digits and the underscore symbol.

    Example: snippet "wishBDay", which you can use to wish your friends Happy Birthday! Since the name of the friend keeps changing, so you can add a placeholder to the snippet body, as "%name%"(without quotes). It would be auto-highlighted whenever you use the snippet and you can type your friend's name into it.

    Usage of a placeholder:

    1. Insert placeholders into the snippet body while creating or editing a snippet. Then, use this snippet.
    2. The first placeholder would be auto-highlighted. Type whatever you want to type there.
    3. Press `[Tab]` key to move to the next placeholder.
    4. Repeat steps 2 and 3 till all placeholders are done filling.

    What is Mathomania?

    Often, mathematical calculations require you to leave your current work and use the calculator. But, using Mathomania, you can do calculations within the text editor itself.

    Supported operations are: addition, subtraction, multiplication, division, exponentiation (of integer constants only), and percentage (using % symbol). You can group expressions using parentheses too.

    Usage:

    1. Type `[[`, the other `]]` pair of braces would auto-complete.
    2. Type the math expression to be evalated inside the braces.
    3. Press `=` key and the result of evaluation would replace the `[[...=]]`!

    What are Date/Time macros?

    Date-Time macros can be used to insert current date and time related data.

    Using Date-Time macros

    They are to be placed inside a snippet body while creating the snippet.

    1. Place `[[%d()]]` inside the snippet body.
    2. Place one or more of the symbols inside the parentheses, from the symbol list, without those quotes.
    3. When you use the snippet, this entire string of `[[%d(...)]]` would be replaced by the current value of the symbols it contains.

    Symbol list

    • 'D' - date (14)
    • 'Do' - date (14th)
    • 'MMMM' - month (March)
    • 'MMM' - month (Mar)
    • 'MM' - numeric month (03)
    • 'YYYY' - year
    • 'hh' - hour (24hour)
    • 'h' - hour (12hour)
    • 'm' - minutes
    • 's' - seconds
    • 'dddd' - day (Sunday)
    • 'ddd' - day (Sun)
    • 'a' - am/pm
    • 'date' - produces current date (March 05, 2015)
    • 'time' - produces current time (01:29:55)

    What are Variables?

    Variables are used to insert dynamic values at any time while typing. They are inbuilt and cannot be created by you. Of course, I would love a recommendation from you for a new variable! (email: prokeys.feedback@gmail.com)

    Usage:

    1. Type `[[`, the other pair of braces `]]` would auto-complete.
    2. Type the variable name inside them, and press `=`.
    3. This entire `[[...=]]` would be replaced by the dynamic values of these variables.

    The following Variables (case-insensitive) are supported (use without quotes):

    1. 'date' - current date (March 05, 2015)
    2. 'time' - current time (01:29:55)
    3. 'version' - current browser version (Chrome 43.0.2357.130)

    How are Variables different from Snippets?
    Variables are more useful when you need to retrieve system-related dynamic values. Like the version of the browser, which would need to be manually entered into a snippet. A Variable would get it automatically.

    Troubleshoot

    Snippet replacement is not working
    • Make sure your caret(cursor) is placed just after the snippet name. Example: If your snippet name is "abcd", place your caret after "d"
    • Snippet names are case-sensitive. "ABCD" and "abcd" are not the same.
    • Refresh the webpage you are working on.
    • If nothing works, then take a backup of your data and reinstall ProKeys. Restore your data back and try again.
    • If it still does not work, contact me at prokeys.feedback@gmail.com
    I get an error reading: "Invalid Snip Name! Please prefix it with a underscore and try again." while creating/editing snips.
    Some words cannot be allowed because they interfere with the internal working of ProKeys. But prefixing them with an underscore will allow you to use them, though you'd need to type the underscore at the start while using it on a webpage. Example, if word 'abcd' gives this error, then use '_abcd' as the snippet name.
